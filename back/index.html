周云蓬，民谣歌者。

1970年出生于辽宁，幼年时，因患眼病随母亲四处求医。整个童年充满了火车、医院、手术室和酒精棉的味道。9岁时彻底失明，15岁开始弹吉他，23岁毕业于长春大学特教学院中文专业。之后，游历了十多个城市，以卖唱为生，并且创办诗歌民刊《命与门》和《低岸》，后只身赴西藏。34岁签约摩登天空，发行首张专辑《沉默如谜的呼吸》，被认为是最具人文气质的中国民谣专辑。目前定居在北京。

ueditor上传图片到七牛(asp.net mvc)

0 Comments

ueditor是百度推出的一款文本编辑器，其内部提供了图片上传的逻辑，可以把本地的图片上传到网站服务器。但是包括本人在内的很多草根站长可能会选择把图片放到其它一些CDN服务器上去，以便节约自己服务器的流量压力。
本人选择的是七牛云存储，因为它提供免费的图片存储服务，且支持外链、图片处理等高级的功能（貌似阿里云也正在推出类似的服务）。
好了，编辑器和图片存储都有了，现在问题是如何把两者结合起来，能否使用ueditor将图片直接上传到七牛，除此之外，在使用ueditor进行图片管理的时候能够直接显示七牛上的图片。带着问题去问度娘和谷哥，找到了一位朋友写的ueditor上传图片到七牛云存储（form api,java），根据这篇文章的方法，基于C#语言，我也同样成功实现了预期的功能

bootstrap-dropdown插件点击显示修改为鼠标悬浮显示

0 Comments

关注Bootstrap很久了，之前在网上下载过几套Bootstrap的后台模板，使用下来效果不错，自己也层按照Bootstrap中国官网bootcss.com的教程自己搭建了一个清爽的后台用于“企业宝项目”。
没用过它搭建过网站前台,最近无意间发现Ucloud的官网也是基于bootstrap的。页面风格蛮喜欢的。查看源代码发现这个js是基于seajs的,js代码download下来很麻烦。于是按照自己的方法来一便,总结如下。
1.css拷贝不是很麻烦。略过…
2.页面轮播图是基于“Unslider”的 .具体内容见官网http://unslider.com/
3.一番工作下来之后，就剩导航条了，这个导航条是基于bootstrap-dropdown的，拷贝官方dropdown.js 导航条显示部分需要点击才显示，不是想要的效果，搜索之后发现https://github.com/ibmsoft/twitter-bootstrap-hover-dropdown 这个源码可以解决这个问题。
